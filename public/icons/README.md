# PWA Icons

You need to create two icon files in this directory:

1. `icon-192x192.png` - 192x192 pixels
2. `icon-512x512.png` - 512x512 pixels

## How to Create Icons

### Option 1: Online Tools
- **PWA Builder Image Generator**: https://www.pwabuilder.com/imageGenerator
- **Real Favicon Generator**: https://realfavicongenerator.net/

### Option 2: Design Tools
- Use Figma, Canva, Photoshop, or any image editor
- Create a square canvas (512x512 recommended)
- Design your app logo/icon
- Export as PNG in both sizes

### Option 3: Quick Placeholder
For testing, you can use a simple colored square or your app's first letter.

## Icon Requirements
- Format: PNG
- Background: Should match your theme (currently black #000000)
- Design: Simple, recognizable, works at small sizes
- Purpose: "any maskable" (works on all platforms including Android adaptive icons)

## Testing
After adding icons, test your PWA installation on:
- Chrome Desktop (chrome://flags - enable PWA installation)
- Chrome Mobile (Android)
- Safari Mobile (iOS - Add to Home Screen)
